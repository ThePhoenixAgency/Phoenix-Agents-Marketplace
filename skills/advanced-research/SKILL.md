# Advanced Research
## Méthode
1. Définir la question de recherche (précise)
2. Identifier 3+ sources indépendantes
3. Cross-valider les informations
4. Synthétiser avec citations
5. Evaluer la fiabilité des sources
## Evaluation des sources
| Critère | Question |
|---------|----------|
| Autorité | Qui est l'auteur ? Qualifications ? |
| Actualité | Date de publication ? A jour ? |
| Objectivité | Biais potentiel ? Sponsorisé ? |
| Exactitude | Corroboré par d'autres sources ? |
## Format de synthèse
```
QUESTION : [question]
SOURCES : [nombre] sources consultées
REPONSE :
- Point 1 [source A, source B]
- Point 2 [source C]
CONFIANCE : Haute/Moyenne/Basse
LIMITES : [ce qui manque]
```
