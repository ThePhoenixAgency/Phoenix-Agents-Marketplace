# Apple Ecosystem
## Platforms
| Plateforme | Framework UI | Notes |
|-----------|-------------|-------|
| iOS | SwiftUI / UIKit | Mobile-first |
| macOS | SwiftUI / AppKit | Desktop, menus, window mgmt |
| watchOS | SwiftUI | Complications, glances |
| tvOS | SwiftUI / UIKit | Focus-based navigation |
| visionOS | SwiftUI + RealityKit | Spatial computing |
## Integration
- iCloud (CloudKit, Key-Value, Documents)
- Keychain (secrets, credentials)
- App Groups (data sharing entre apps)
- Widgets (WidgetKit)
- Shortcuts (Intents)
- Universal Links
- Handoff & Continuity
## App Store
- App Store Review Guidelines
- In-App Purchase (StoreKit 2)
- TestFlight pour la beta
- App Privacy labels
