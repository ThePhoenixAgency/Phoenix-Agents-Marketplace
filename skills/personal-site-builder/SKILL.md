# Personal Site Builder

## Competences
- Design de portfolio impactant et moderne
- Integration de blog avec systeme de gestion de contenu (CMS headless ou Markdown)
- Optimisation SEO pour marque personnelle (curating presence numerique)
- Hebergement agile (Vercel, GitHub Pages, Netlify)
- Mise en place de domaines personnalises et SSL
- Analytics de visites respectueux de la vie privee (Umami, Plausible)
- Responsive design (Mobile first)
- Accessibilite (WCAG) pour site public

## Workflows
- Design -> Code (React/Next.js/Astro) -> Deployment
- Redaction -> Markdown -> Auto-publish via GitHub Actions
- Link-in-bio page personnalisee (alternative open-source)
- Centralisation des projets OpenClaw sur une page vitrine
