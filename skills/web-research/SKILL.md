# Web Research
## Methode
1. Definir la question de recherche (precise)
2. Identifier 3+ sources independantes
3. Cross-valider les informations
4. Synthetiser avec citations
5. Evaluer la fiabilite des sources
## Evaluation des sources
| Critere | Question |
|---------|----------|
| Autorite | Qui est l'auteur ? Qualifications ? |
| Actualite | Date de publication ? A jour ? |
| Objectivite | Biais potentiel ? Sponsorise ? |
| Exactitude | Corrobore par d'autres sources ? |
## Format de synthese
```
QUESTION : [question]
SOURCES : [nombre] sources consultees
REPONSE :
- Point 1 [source A, source B]
- Point 2 [source C]
CONFIANCE : Haute/Moyenne/Basse
LIMITES : [ce qui manque]
```
