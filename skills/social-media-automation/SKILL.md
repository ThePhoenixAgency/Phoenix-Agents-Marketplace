# Social Media Automation
## Scheduling workflow
```
1. Creer le contenu (texte + visuels)
2. Adapter par plateforme (format, hashtags, ton)
3. Programmer la publication (heures optimales)
4. Publier automatiquement
5. Monitorer l'engagement
6. Repondre aux commentaires
7. Analyser les performances
```
## Heures optimales (indicatif)
| Plateforme | Meilleurs creneaux |
|-----------|-------------------|
| LinkedIn | Mar-Jeu 8h-10h |
| X/Twitter | Lun-Ven 12h-15h |
| Instagram | Mar-Ven 11h-13h |
| TikTok | Jeu-Sam 19h-21h |
## Analytics cles
- Reach / Impressions
- Engagement rate (likes + comments + shares / reach)
- Click-through rate (CTR)
- Follower growth rate
- Best performing content type
