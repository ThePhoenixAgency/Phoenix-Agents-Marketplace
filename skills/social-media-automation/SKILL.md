# Social Media Automation

## Plateformes

| Plateforme | API | Content types |
|-----------|-----|---------------|
| X/Twitter | v2 API | Tweets, threads, polls, spaces |
| LinkedIn | Marketing API | Posts, articles, newsletters, carousels |
| Instagram | Graph API | Posts, stories, reels, carousels |
| TikTok | Content API | Videos, duets, stitches |
| YouTube | Data API v3 | Videos, shorts, community posts |
| Reddit | API | Posts, comments, AMAs |
| Product Hunt | API | Launches, collections |
| Facebook | Graph API | Posts, stories, groups |

## Workflow complet
```
1. Strategie (calendrier editorial, objectifs)
2. Creation contenu (texte + visuels + video)
3. Adaptation par plateforme (format, hashtags, ton)
4. Programmation (heures optimales par plateforme)
5. Publication automatique (cross-posting)
6. Engagement (reponses, commentaires, DMs)
7. Analytics et reporting
8. Optimisation (A/B testing, iteration)
```

## Analytics cles
- Reach / Impressions
- Engagement rate (likes + comments + shares / reach)
- Click-through rate (CTR)
- Follower growth rate
- Conversion rate
- Best performing content type
- ROI par campagne

## Growth tactics
- Hashtag strategy (niche + trending)
- Collaboration et cross-promotion
- User-generated content
- Viral hooks et storytelling
- Community building (DMs, replies, groups)
- Paid amplification (minimum viable budget)
