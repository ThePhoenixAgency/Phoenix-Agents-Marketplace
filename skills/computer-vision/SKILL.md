# Computer Vision

## Competences
- Image-to-text (description detaillee d'images et de scenes)
- Object detection et classification (identification de milliers de categories)
- Face detection et analyse d'attributs (sans stockage de biometrie)
- OCR in-the-wild (lecture de texte sur des panneaux, panneaux de rue, etc.)
- Visual search (trouver des objets similaires)
- Image captioning pour l'accessibilite
- Analyse de sentiments via expressions faciales
- Pose detection (points de repere du corps humain)

## Cas d'usage
- Analyse de captures d'ecran pour support technique automatique
- Surveillance visuelle pour la securite (detection de mouvement, intrusion)
- Inventaire automatique par reconnaissance d'objets
- Aide a la navigation pour les malvoyants
- Extraction d'informations de graphiques et diagrammes
