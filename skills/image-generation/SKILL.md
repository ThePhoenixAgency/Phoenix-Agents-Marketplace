# Image Generation

## Competences
- Text-to-image (Stable Diffusion, DALL-E, Midjourney via API)
- Logo creation (vectoriel, variations, formats multiples)
- Banniere creation (social media sizes, Open Graph)
- Brand identity kit (couleurs, typographie, icones)
- Product mockups
- Image editing (inpainting, outpainting, upscaling)
- Background removal
- Style transfer et consistency across assets
- Batch generation avec templates

## Formats de sortie
- PNG, SVG, WebP, AVIF
- Multi-resolution (1x, 2x, 3x pour mobile)
- Format par plateforme (Twitter 1200x675, Instagram 1080x1080, etc.)
