# Copywriting
## Formules
- AIDA : Attention -> Interest -> Desire -> Action
- PAS : Problem -> Agitate -> Solution
- BAB : Before -> After -> Bridge
## Regles
- 1 idee = 1 phrase
- Actif > passif
- Concret > abstrait
- Benefice > fonctionnalite
- Call to action clair et unique
