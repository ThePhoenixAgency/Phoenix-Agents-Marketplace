# Document Processing

## Competences
- OCR (Tesseract, Apple Vision, Google Document AI)
- Extraction de donnees structurees (factures, contrats, CV)
- Classification automatique de documents
- Generation de PDF, Word, Excel
- Template engine (Mustache, Handlebars, docx-templater)
- Signature electronique (DocuSign API, eIDAS)
- Archivage et indexation full-text
- Conversion entre formats (PDF -> DOCX, HTML -> PDF)
- Metadata extraction (EXIF, PDF properties)
- Watermarking et protection

## Workflows
- Scan -> OCR -> Classification -> Extraction -> Base de donnees
- Template -> Remplissage -> Validation -> Signature -> Archivage
- Document entrant -> Tri automatique -> Routage vers agent competent
