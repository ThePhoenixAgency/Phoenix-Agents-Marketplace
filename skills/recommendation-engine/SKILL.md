# Recommendation Engine

## Competences
- Vector embeddings pour representaion semantique
- Similarity search (cosine, dot product, euclidean)
- Collaborative filtering (user-based, item-based)
- Content-based filtering (attributs et metadata)
- Hybrid models (collaborative + content)
- Re-ranking et personalization
- A/B testing de modeles de recommandation
- Cold start strategies (nouveaux utilisateurs, nouveaux items)

## Architecture
- Embedding server (generation de vecteurs)
- Vector store (index pour recherche rapide)
- API de recommandation (REST/GraphQL)
- Pipeline d'ingestion (batch + real-time)
- Feedback loop (click, achat, rating -> re-training)

## Applications
- Recommandation de produits / contenu
- Search semantique (recherche en langage naturel)
- Matching (candidats/emplois, utilisateurs/projets)
- Deduplication par similarite semantique
