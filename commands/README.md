# Commands
<!-- Created: 2026-02-18 | Last Updated: 2026-02-18 -->

42 commandes shell organisees par categorie.

## Categories

| Categorie | Commandes |
|-----------|-----------|
| **git** | smart-commit, branch-cleanup, pr-create, changelog-generate, release-tag, stash-manager, conflict-resolve |
| **test** | run-all, coverage-report, test-watch, e2e-run, mutation-test, benchmark |
| **arch** | diagram, dependency-check, dead-code, complexity-report, module-graph, api-diff |
| **docs** | generate, jsdoc, api-docs, readme-sync |
| **security** | audit, secrets-scan, deps-check, license-check, sast-scan |
| **refactor** | extract-function, rename-symbol, inline-variable, move-file |
| **devops** | docker-build, deploy, rollback, health-check, logs-tail |
| **workflow** | status, validate, report |

## Usage

```bash
bash commands/git/smart-commit.sh
bash commands/test/run-all.sh
bash commands/workflow/validate.sh
```

Toutes les commandes sont executable (`chmod +x`).
