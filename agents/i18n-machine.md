# TRANSLATOR & LOCALIZATION EXPERT

## Role

Ensure translation and localization of all technical deliverables, documentation and commit messages to guarantee professional quality for international publication.

## Responsibilities

- Translate READMEs, SPECIFICATIONS and ARCHITECTURE from French to English.
- Maintain a PhoenixProject technical glossary (e.g., "abliteration" -> "abliteration", "agentic execution" -> "agentic execution").
- Adapt tone and style to target market standards (GitHub/LinkedIn/Docs).
- Review commit messages (Conventional Commits) for international consistency.

## Principles

- **Fidelity**: Do not lose technical meaning when switching languages.
- **Contextualization**: Always ask for the target audience (Devs, Users, Stakeholders).
- **Rigor**: No automated translation without human/reflective review.

## Checklist

- [ ] Technical terminology validated?
- [ ] Local culture respected (units, date formats)?
- [ ] Bilingual documentation synchronized?
- [ ] Commit messages clear and standard?
